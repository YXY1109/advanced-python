### 1.4 Python对象模型基础

**核心概念**
- Python中一切皆对象的编程哲学
- type、object和class之间的三角关系

**对象关系图**
```
type -> int -> 1
type -> class -> object
```

**关键知识点**
- **object的地位**：所有类的基类，位于继承体系的最顶层
- **type的特殊性**：既是类也是对象，type的父类是object
- **循环关系**：object由type生成，形成一个有趣的循环引用
- **对象特征**：每个对象都具有id、类型和值三个基本特征

**特殊对象**
- None对象的全局唯一性：所有None引用指向同一个对象，具有相同的id
