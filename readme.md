# Python3高级核心技术学习仓库

## 项目概述

本仓库是Python高级编程课程的完整学习资料库，系统涵盖Python高级编程和并发编程核心技术。课程由Bobby主讲，面向有Python基础并希望提升至高级工程师水平的学习者。

## 🎯 学习目标

- 深入理解Python的设计理念和底层原理
- 掌握Python高级面向对象编程技术
- 精通多线程、多进程和异步编程
- 具备开发高并发应用的核心能力
- 通过Python高级工程师技术面试

## 📚 课程大纲

### 第一部分：Python对象模型与底层原理
- **第1章：导学与环境配置** - 课程介绍、开发环境搭建
- **第2章：Python一切皆对象** - type、object、class关系、内置类型
- **第3章：魔法函数** - Python数据模型、特殊方法详解
- **第4章：面向对象高级编程** - 鸭子类型、抽象基类、元类、属性管理

### 第二部分：数据结构与内存管理
- **第5章：序列类型深入** - 可切片对象、bisect维护、推导式
- **第6章：字典与集合** - 哈希表原理、性能优化
- **第7章：内存管理** - 变量本质、垃圾回收、参数传递

### 第三部分：函数式编程与生成器
- **第8章：元类编程** - 自定义元类、ORM实现、属性描述符
- **第9章：迭代器与生成器** - 迭代协议、生成器原理、大文件处理

### 第四部分：网络编程基础
- **第10章：Socket编程** - HTTP/TCP协议、客户端-服务器模型

### 第五部分：并发编程核心
- **第11章：多线程与多进程** - GIL原理、线程同步、进程通信
- **第12章：异步编程基础** - IO多路复用、协程概念、事件循环
- **第13章：AsyncIO实战** - aiohttp高并发爬虫、asyncio应用
- **第14章：课程总结** - 知识体系梳理、实践指导

## 🛠️ 技术栈与环境要求

### 核心技术
- **Python 3.x** - 主要编程语言
- **Whisper** - 语音识别与转文本
- **MoviePy** - 视频音频处理
- **OpenAI API** - 文本总结与生成
- **AsyncIO** - 异步编程框架

### 环境配置
```bash
# 安装依赖
pip install -r requirements.txt

# 主要依赖包
numpy==1.26.4
moviepy==2.1.2
openai==1.72.0
openai-whisper==20240930
```

### 环境变量配置
创建`.env`文件：
```env
OPENAI_BASE_URL=https://openrouter.ai/api/v1
OPENAI_API_KEY=your_api_key_here
```

## 📖 学习资料说明

### 文档类型
1. **个人学习笔记** (`readme.md`) - 手工整理的核心知识点
2. **自动生成文档** (其他md文件) - 通过AI辅助生成的课程总结

### 自动化内容生成
项目使用`video_to_md.py`实现视频内容自动化处理：

```python
# 工作流程
视频文件 → 音频提取 → 语音转文本 → AI总结 → Markdown文档
```

**技术实现：**
- MoviePy提取视频音频
- Whisper模型进行语音识别（large-v3-turbo模型）
- OpenAI API进行内容总结（Qwen模型）
- 生成结构化Markdown文档

### 使用方法
```bash
# 处理files目录中的所有视频文件
python video_to_md.py
```
