## 鸭子类型的理解

```
python里的鸭子类型是指，只要class里实现了特定的魔法函数，就可以被认为是某种类型。


instance 可以判断出继承关系的
type 不可以判断继承关系的

继承关系是mro，实现是c3算法

私有属性：
    1. 私有属性：__name，变形为：_class__name，没有绝对安全，规范代码
    2. 私有方法：__name()
    

python的自省机制：通过一定的机制查询到对象的内部结构
    1. dir()：列表，所有属性
    2. __dict__：实例的属性
    
    
super()是调用mro的下一个类，而不是父类

多继承的经验，设计不好，会导致继承关系混乱，导致代码维护困难，建议单继承或者使用mixin模式
例如：django中的orm，使用mixin模式，功能单一，解耦，通过任意组合实现功能

上下文管理器协议

__exit__()
```
