## 📖 章节内容详解

### 2.1 Python中一切皆对象

**核心理念**
- Python是比Java更加彻底的面向对象编程语言
- 在Python中，不仅是类的实例是对象，类本身、函数等都是对象
- 这种设计理念赋予Python极高的动态性和灵活性

**对象的特性**
- **一等公民地位**：Python中的对象享有完全的语言特性支持
- **动态赋值**：对象可以被赋值给变量
- **集合存储**：对象可以添加到列表、字典等集合中
- **函数参数**：对象可以作为函数参数传递
- **函数返回值**：对象可以作为函数的返回值

**实际应用**
- 函数作为对象使用的具体实例
- 类作为对象操作的详细说明
- 函数返回函数的概念（装饰器的基础原理）

### 2.2 type、object和class之间的关系

**三大核心概念**

**1. 对象（Object）**
- Python中的一切都是对象
- `object`是所有类的基类，位于继承体系的最顶层
- 所有类都直接或间接继承自`object`
- `object`没有基类，是继承体系的根节点

**2. 类（Class）**
- 类是创建对象的模板
- 在Python中，类本身也是对象
- 类是由`type`类生成的实例
- 定义类的过程实际上就是创建`type`实例的过程

**3. 类型（Type）**
- `type`是用于创建其他类的特殊类
- `type`本身也是类，同时是自身的实例
- `type`继承自`object`，但也能创建`object`类
- 形成了一个有趣的闭环关系

**关系图谱**
```
type -> class -> object
↑         ↖        ↙
  └────── type ←────┘
```

**关键逻辑关系**
- `type`用于创建类
- 所有类继承自`object`
- `object`是`type`的实例
- `type`是自身的实例
- 形成了一个完整的对象创建体系

### 2.3 Python中的内置类型

**对象的三大特征**

**1. 身份（Identity）**
- 对象在内存中的地址
- 通过`id()`函数获取
- 每个对象都有唯一的身份标识

**2. 类型（Type）**
- 每个对象都属于特定类型
- 如`int`、`str`、`list`等
- 类型决定了对象的属性和方法

**3. 值（Value）**
- 对象的具体内容
- 如数字`1`是`int`类型对象的值
- 字符串内容是`str`类型对象的值

**Python内置类型分类**

**1. 数值类型**
- `int`（整型）
- `float`（浮点型）
- `complex`（复数型）

**2. 布尔类型**
- `bool`（布尔型）

**3. 迭代类型**
- 迭代器（Iterator）
- 生成器（Generator）
- 用于循环遍历数据

**4. 序列类型**
- `list`（列表）
- `tuple`（元组）
- `range`（范围对象）
- `str`（字符串）

**5. 二进制序列**
- `bytes`（字节）
- `bytearray`（可变字节）
- `memoryview`（内存视图）

**6. 映射类型**
- `dict`（字典）
- 用于存储键值对

**7. 集合类型**
- `set`（集合）
- `frozenset`（不可变集合）

**8. 特殊类型**
- 模块（Module）
- 类（Class）
- 函数（Function）
- 方法（Method）
- 代码对象（Code Object）
- `object`（根对象）
- `type`（类型类）
- `ellipsis`（省略号对象）
- `NotImplemented`（未实现对象）

**9. 上下文管理类型**
- 用于`with`语句的对象
- 实现上下文管理协议的对象
